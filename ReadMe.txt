Our program doesn't have an extension file. So you have to run the code Antivirus.asm in emu8086.
Please follow the instruction:
1. Delete the vdrive directory from the emu8086 directory in the c drive of your PC and create as many shortcut files as you like in all the directories and subdirectories.
2. copy and paste the given vdrive folder inside "Extras" folder.
3. copy and paste the directory.txt file inside "Extras" folder" at the mybuild folder in the emu8086 directory in the C drive of your PC
4. Run the Antivirus.asm in emu8086.
